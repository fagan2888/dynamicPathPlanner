## Dependencies
- ROS
- Numpy
- Matplotlib

## Build Instructions
Open a terminal and type in the following,
```
mkdir <workspace>
cd <workspace>
mkdir src
cd src
git clone https://github.com/Pruthvi-Sanghavi/dynamicPathPlanner.git
cd ..
catkin_make
```
